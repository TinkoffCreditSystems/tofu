title: Installation
---

The library consists of a few modules, which can be installed all at once:  
`"ru.tinkoff" %% "tofu" % tofu-version`  
or as standalone dependencies:

## Core
`"ru.tinkoff" %% "tofu-core" % tofu-version`  

## Concurrent
`"ru.tinkoff" %% "tofu-concurrent" % tofu-version`  

## Config
`"ru.tinkoff" %% "tofu-config" % tofu-version`  

## Env
`"ru.tinkoff" %% "tofu-env" % tofu-version`  

## Logging
`"ru.tinkoff" %% "tofu-logging" % tofu-version`  

## Memo
`"ru.tinkoff" %% "tofu-memo" % tofu-version`  

## Optics
`"ru.tinkoff" %% "tofu-optics-core" % tofu-version`  
`"ru.tinkoff" %% "tofu-optics-macro" % tofu-version`  
`"ru.tinkoff" %% "tofu-optics-interop" % tofu-version`
